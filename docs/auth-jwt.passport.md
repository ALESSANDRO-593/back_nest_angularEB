# instalacion 
```
npm i --save @nestjs/jwt passport-jwt brcypt
npm i - - save-dev @types/passport-jwt
```

# arrancamos el proyecto
```
npm run start:dev
```
# modules y resource
```
nest g mo modules/auth 
nest g mo modules/auth 
```
# controllers y services
```
nest g co modules/auth   
nest g s modules/auth   
```
